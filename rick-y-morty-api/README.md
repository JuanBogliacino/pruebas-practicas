!! REEMPLAZAMOS LA API DE PELICULAS POR LA DE RICK Y MORTY

**Pregunta:**
Se te ha dado una API que proporciona información sobre personajes de Rick Y Morty. Debes crear una aplicación web en React que muestre un listado de personajes y permita a los usuarios hacer clic en un personaje para ver más detalles. Debes implementar al menos dos vistas: una lista de personajes y una vista de detalles del personaje. Utiliza la API proporcionada para obtener los datos de los personajes.

**Instrucciones:**
- Utiliza la siguiente URL de la API para obtener la lista de personajes: `https://rickandmortyapi.com/api/character`
- Crea un componente `CharacterList` que muestre la lista de personajes.
- Crea un componente `CharacterDetail` que muestre los detalles de un personaje cuando se selecciona.
- Implementa la navegación entre la lista de personajes y los detalles utilizando React Router u otra solución similar.

Esta pregunta evalúa la capacidad del desarrollador para consumir una API, crear componentes React reutilizables, implementar navegación entre vistas y presentar información de manera efectiva. Las pruebas técnicas varían en enfoque y complejidad, pero esta pregunta podría ser un buen desafío para evaluar las habilidades de desarrollo frontend con React.
